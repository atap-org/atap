    1. Add new store interfaces to api.go:
       ```go
       type ClaimStore interface {
           CreateClaim(ctx context.Context, c *models.Claim) error
           GetClaimByCode(ctx context.Context, code string) (*models.Claim, error)
           RedeemClaim(ctx context.Context, code string, redeemedBy string) error
       }

       type DelegationStore interface {
           CreateDelegation(ctx context.Context, d *models.Delegation) error
       }

       type PushTokenStore interface {
           UpsertPushToken(ctx context.Context, entityID, token, platform string) error
           GetPushToken(ctx context.Context, entityID string) (*models.PushToken, error)
           DeletePushToken(ctx context.Context, entityID string) error
       }
       ```

    2. Extend Handler struct with new fields: claimStore ClaimStore, delegationStore DelegationStore, pushTokenStore PushTokenStore, pushService *push.PushService (can be nil). Update NewHandler to accept these.

    3. Create `claims.go` with two endpoints:

       `CreateClaim` (POST /v1/claims) -- authenticated (agent creates an invite):
       - Get authenticated entity from c.Locals("entity")
       - Generate claim ID (crypto.NewClaimID()) and code (crypto.GenerateClaimCode())
       - Set expires_at to 24 hours from now
       - Save via claimStore.CreateClaim
       - Return 201 with {id, code, link: "https://link.atap.app/claim/{code}"}

       `GetClaim` (GET /v1/claims/:code) -- public (app checks claim validity):
       - Get claim by code via claimStore.GetClaimByCode
       - Return 404 if not found, return claim status and creator info

    4. Create `human.go` with RegisterHuman endpoint:

       `RegisterHuman` (POST /v1/register/human) -- public (no auth, this IS registration):
       - Parse HumanRegisterRequest (public_key, email, claim_code)
       - Validate claim code via claimStore.GetClaimByCode -- must exist and be 'pending'
       - Decode public key from base64
       - Derive human ID: crypto.DeriveHumanID(publicKeyBytes)
       - Generate key ID: crypto.NewKeyID(publicKeyBytes)
       - Create entity: type="human", id=derivedID, uri="human://{derivedID}"
       - Save entity via entityStore.CreateEntity
       - Redeem claim via claimStore.RedeemClaim(code, entity.ID)
       - Create delegation: crypto.NewDelegationID(), delegator=human, delegate=claim.CreatorID (the agent)
       - Save delegation via delegationStore.CreateDelegation
       - Return 201 with {entity (public view), key_id}
       - Email verification is deferred to a follow-up (store email as attestation metadata, but don't block registration on verification)

    5. Create `push.go` with push token endpoint:

       `RegisterPushToken` (POST /v1/entities/:entityId/push-token) -- authenticated:
       - Verify authenticated entity matches URL entityId (own-entity enforcement)
       - Parse {token: string, platform: "android"|"ios"}
       - Validate platform is "android" or "ios"
       - Upsert via pushTokenStore.UpsertPushToken
       - Return 204 No Content

    6. Register routes in SetupRoutes:
       - POST /v1/claims (auth required)
       - GET /v1/claims/:code (public)
       - POST /v1/register/human (public)
       - POST /v1/entities/:entityId/push-token (auth required)
