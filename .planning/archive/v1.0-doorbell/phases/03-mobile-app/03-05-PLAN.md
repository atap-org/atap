       - CreateClaim stores in map, GetClaimByCode looks up by code, RedeemClaim updates status
       - CreateDelegation stores in map
       - UpsertPushToken stores/updates in map, GetPushToken looks up

    2. Update test handler setup to pass new fake stores to NewHandler.
